# A-6 Digital Tools Buying Website

## Project Description
This is a responsive digital tools buying website where users can browse products, add them to the cart, and proceed to checkout. The project includes dynamic product display, cart functionality, and interactive alerts using React-Toastify.

## Technologies Used
- **React.js**  
- **Tailwind CSS** & **DaisyUI**  
- **JavaScript (ES6+)**  
- **React-Toastify**  
- **JSON** (for product data)

## Main Features
1. **Product Browsing & Cart Toggle**  
   - Users can view products in a 3-column layout and toggle between product and cart sections.

2. **Interactive Cart Functionality**  
   - Add, remove products, view total count in navbar, and proceed to checkout to clear cart items.

3. **Responsive Design & Alerts**  
   - Fully responsive for mobile, tablet, and desktop.  
   - Alerts for add to cart, remove, and checkout using React-Toastify.
